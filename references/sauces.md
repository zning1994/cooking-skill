# Sauces

This file provides practical flavor profiles and ratio baselines for home cooking.

Scope: only core sauce patterns that support frequent user requests.

## Usage rules

1. Ratios are starting points, not rigid formulas.
2. State flavor intent before listing ratio.
3. Adjust by taste and dish moisture.
4. Avoid forcing sauce complexity into simple dishes.

## Core profiles

### Light savory stir-fry

Use for: quick vegetable/protein stir-fry

Baseline:

- light soy: 1
- oyster sauce: 0.5
- sugar: a small pinch
- water/stock: 1-2

### Braise / red-braise base

Use for: chicken thigh, pork belly, potato braise routes

Baseline:

- light soy: 2
- dark soy: 0.5
- sugar: 0.5-1
- cooking wine: 1
- water/stock: enough to braise

### Sweet-sour base

Use for: sweet-sour home dishes

Baseline:

- sugar: 2
- vinegar: 2
- light soy: 1
- ketchup (optional): 1
- water: 2

Adjustment:

- too sharp: add sugar/water gradually
- too flat: add a little vinegar late

### Spicy-savory base

Use for: 下饭 dishes with chili-bean profile

Baseline:

- doubanjiang/chili paste: 1
- light soy: 1
- sugar: 0.3-0.5
- water/stock: 2-3

### Garlic-forward base

Use for: 蒜香 stir-fry / sauce finish

Baseline:

- minced garlic: generous
- light soy: 1
- oyster sauce: 0.5
- sugar: pinch
- water: 1-2

## Thickening guidance

If sauce should coat ingredients:

- add light starch slurry near the end
- keep heat active
- stop once glossy, avoid paste texture

## Fast fixes

### Too salty

- add neutral bulk or water where suitable
- rebalance before adding sugar

### Too sweet

- add acidity or soy depending on profile

### Too sour

- extend cooking
- add small sugar increments

### Too thin

- reduce first
- then small slurry if dish type fits
