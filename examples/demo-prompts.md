# Demo Prompts

这些 prompt 适合：

- README 演示
- 社交媒体发帖
- 朋友第一次试用
- 后续补 evals 时继续扩展

---

## 想吃什么

```text
今晚吃啥
今晚想吃点热乎的
想吃个下饭的，别太复杂
两个人吃，想来个家常硬菜
今天下雨，想喝点汤汤水水的
```

## 家里有什么

```text
冰箱里有鸡腿土豆洋葱，能做什么
只有番茄鸡蛋挂面，晚饭怎么解决
家里有豆腐、青椒、肉末，出一道菜
剩米饭、鸡蛋、火腿，怎么做别太敷衍
我这边有白菜、粉条、五花肉，来个稳点的
```

## 指定菜 / 指定技法

```text
宫保鸡丁怎么做
番茄炒蛋怎么做味道更顺
牛肉怎么炒嫩一点
糖醋汁怎么调比较稳
土豆丝怎么炒才脆
鸡腿怎么做不柴
```

## 替代方案 / 救场

```text
没有料酒怎么办
没有淀粉怎么救
做咸了怎么补
番茄太酸怎么调回来
没有蚝油能不能做这个菜
没有炒锅，普通平底锅怎么做
```

## 边界测试

```text
给我做一桌 10 人年夜饭
帮我做减脂高蛋白一周精确食谱
法餐牛排酱汁体系怎么搭
提拉米苏和可颂怎么做
```

这些问题也可以问，但它们不是 `cooking.skill` 当前最强的部分。
